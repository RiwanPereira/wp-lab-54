Introduction

Flexbox is a layout model designed for one-dimensional layouts.

It simplifies spacing and alignment.

display: flex;
  • Easy layouts
Flex Container

The container defines the layout direction.

It controls wrapping behavior.

flex-direction: row;
  • Row and column
Flex Items

Flex items live inside the container.

They can grow or shrink.

flex: 1;
  • Dynamic sizing
Alignment

Flexbox excels at alignment.

Both axes are controllable.

justify-content: center;
  • Main axis
Responsive Design

Flexbox helps build responsive layouts.

Media queries enhance flexibility.

@media (max-width: 768px)
  • Mobile friendly
